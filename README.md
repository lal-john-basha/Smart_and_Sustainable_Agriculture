
# 🌱 Smart & Sustainable Agriculture System

A **Smart Agriculture System** that monitors environmental conditions and helps farmers make intelligent irrigation decisions using simulated IoT sensor data.

This project demonstrates how **technology, IoT, and data monitoring** can improve farming efficiency and reduce water wastage.

---

# 📌 Project Overview

The Smart & Sustainable Agriculture system simulates a farm monitoring environment where sensors collect real-time data such as:

* Soil moisture
* Temperature
* Irrigation requirement

The system processes this data and determines whether irrigation is required.

The interface is built using **Streamlit**, allowing users to visualize sensor readings and irrigation decisions easily.

# ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/lal-john-basha/Smart_and_Sustainable_Agriculture.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Smart_and_Sustainable_Agriculture
```

### 3️⃣ Install Required Libraries

```bash
pip install streamlit
```

### 4️⃣ Run the Application

```bash
streamlit run app.py
```
